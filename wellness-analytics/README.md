# Aura — AI-Powered Wellness Analytics Platform

Mood Journal → Sentiment Analysis → Stress Trend Detection → Personalized Recommendations →
Dashboard → Weekly Reports → User Analytics.

A B2B corporate-wellness product integrating **NLP, ML, backend, frontend, database, product
thinking, business metrics, and engagement analysis** in one runnable system.

## Run it (pick one)

**Docker — full stack (Postgres + API), one command:**
```bash
docker compose up --build
```

**Local — no Docker (SQLite, zero setup):**
```bash
./run.sh
```

Then open **http://localhost:8000**. Tests: `pytest -q`.
